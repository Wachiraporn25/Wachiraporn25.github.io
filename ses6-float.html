การจัดการเลย์เอาท์บนเว็บเพจด้วย Float

    float กำหนดให้อิลิเมนต์สามารถลอยอยู่ด้านใดด้านหนึ่งโดยกำหนดได้ คือ
  • laft, right
  • inherit ลอยตาม parent element
  • none ไม่ให้มีการลอย (dafault)
    clear ไม่อนุญาติให้มีการลอยของอิลิเมนต์ค่าที่กำหนดได้คือ
  • left, right (ไม่อนุญาติให้มีการลอยทางด้านซ้ายและขวา)
  • both ไม่อนุญาติให้ลอยทั้งสองด้าน
